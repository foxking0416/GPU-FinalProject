GPU-FinalProject

11/24/2014, Update

Alpha Version Demo:  http://foxking0416.github.io/GPU-FinalProject/

####Alpha Features:

    - Spiral line connections (basic)
    - Country Obj (test)
    - Area of study Obj (test)
* 2D UI
  - D3.js, html & css
  - Features for alpha:
    - history timeline
    - selected country name
